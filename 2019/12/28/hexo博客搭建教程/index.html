hexo博客搭建教程

安装hexo博客的步骤：

  • 安装Node.js:

    官网下载node.js，一路next安装就完事了。

    安装完成后会有两个组件：一个是node.js本身，一个是npm包库。

    安装完成了在自己电脑Win+R输入cmd打开控制台输入node -v查看node版本，输入npm -v查看npm版本，显示出来则表示安装成功。

  • 添加国内镜像源：

    因为在国内使用npm下载通常速度都是非常慢的，如果不能科学上网的话将npm换成淘宝的镜像源。

    在控制台输入npm config set registry https://registry.npm.taobao.org

    安装后

  • 注册GitHub账号

  • 安装Git

    Git是一个工具，代码管理工具。GitHub是基于git打造的社区。hexo博客要与GitHub连接，将内容提交到GitHub上去，所以Git安装是必须的。

    直接去Git官网https://www.git-scm.com/download/下载最新版本的Git。

    安装完毕后要进行Git的环境配置：

    在桌面右键打开GitBash，这个就和cmd控制台一样的。

    1. 在GitBash输入：

      git config –global user.name “你注册GitHub时的账户名”

      git config –global user.email “你注册GitHub时用的邮箱”

    2. 生成ssh：

      继续刚才的操作，命令框输入ssh-keygen -t rsa，然后连敲三次回车键，结束后去系统盘目录下（一般在 C:\Users\你的用户名.ssh）(mac: /Users/用户/.ssh）查看是否有。ssh文件夹生成，此文件夹中以下两个文件，如图：

      将ssh文件夹中的公钥（ 打开id_rsa.pub）添加到GitHub管理平台中，在GitHub的个人账户的设置中找到如下界面。打不开命令框输入cat ~/.ssh/id_rsa.pub获取。

      title随便起一个，将公钥（ id_rsa.pub）文件中内容复制粘贴到key中，然后点击Ass SSH key就成了。

      完成后测试一下配置是否成功，在Git Bush命令框（就是刚才配置账号和邮箱的命令框）中继续输入以下命令，回车，出现自己的名字就成功了。

安装Git以及环境配置参考文章：https://blog.csdn.net/huangqqdy/article/details/83032408

  • 正式安装hexo

    • 安装hexo博客并初始化生成静态界面

      1. 新建一个存放hexo文件的文件夹，在GitBash输入mkdir **（文件夹名），输入cd *就是进入文件夹。

      2. 在自己存放hexo的根目录下右键打开GitBash，输入npm i hexo-cli -g
        安装好了后输入hexo -v查看是否成功。

      3. 安装插件：npm install

      4. 初始化hexo博客：输入hexo init 初始化博客，初始化好了后在自己创建的文件夹中可以看到生成的hexo文件。

      5. 打开博客：hexo s打开博客，会出现一个叫 localhost：400的网址，在浏览器打开则能进入博客。（这个暂时只能你自己看，要想被他人访问必须部署到远端。），hexo自动生成一篇名字叫Hello World的博客。

      6. 各个指令：hexo s启动博客，hexo g生成博客 ，ctrl+c关闭本地服务器。

  • 写博客

    写博客是使用markdown格式来写博客的，下载一款md编写软件，推荐Typora。在git bash输入hexo n “文章名字” ，然后打开hexo根目录，打开source文件夹里面包含的_posts文件夹，找到新建的文章进去编辑。然后再GitBash输入 hexo g 生成，然后再输入 hexo s 在本地浏览，没问题后 ctrl+c 退出后再输入hexo d 部署到GitHub仓库里，这样自己和其他人都可以浏览文章了。

  • 更改hexo博客的主题

    1. 本次使用的主题以https://github.com/litten/hexo-theme-yilia主题为示范。(输入指令中间有空格，都有空格，注意看仔细！)

      在GitBash输入指令 git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

      回车后就会向 themes/yilia这个目录克隆，打开hexo根目录可以看到。

      下载速度慢可以在码云注册，然后新建仓库再导入仓库。成功后再下载，速度很快。

      参考文章：https://www.jianshu.com/p/e9cd66d2116f
